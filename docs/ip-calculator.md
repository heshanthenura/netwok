### 1. IP Calculator

This IP Calculator provides essential information about IP addressing and subnetting based on a given IP and subnet mask.

With this tool, you can view the following:

- #### 1. IP Address  
- #### 2. Class  
- #### 3. Subnet Mask  
- #### 4. Netbits (Prefix)  
- #### 5. Hostbits  
- #### 6. Network Address  
- #### 7. Broadcast Address  
- #### 8. First Usable IP  
- #### 9. Last Usable IP  
- #### 10. Total Addresses  
- #### 11. Usable Addresses
